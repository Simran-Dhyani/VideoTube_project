# 🚀 Learnify

Learnify is a modern learning platform that helps users watch educational videos, create timestamp-free notes, and organize their learning journey in one place.

Built using the MERN Stack, Learnify combines video-based learning with personal note management to improve productivity and knowledge retention.

## ✨ Features

### 🔐 Authentication

* User Registration & Login
* JWT Authentication
* Secure Access & Refresh Tokens
* Protected Routes
* Persistent User Sessions

### 🎥 Video Learning

* Watch YouTube videos directly inside the application
* Dedicated Watch Page for focused learning
* Responsive video player experience

### 📝 Smart Notes

* Create notes while watching videos
* View all notes linked to a video
* Edit existing notes
* Delete notes
* Notes stored securely per user

### 📚 My Notes Dashboard

* Automatically groups notes by video
* Displays all videos for which notes exist
* Shows note count for each video
* Quick navigation back to the learning session

### 🎨 Modern UI

* Responsive design using Tailwind CSS
* Shadcn UI Components
* Framer Motion Animations
* Glassmorphism Effects
* Premium Dark Theme Experience
